----------------------------------------------COLORS LAB----------------------------------------------

-Description:

This is the COLORS application, created in the COP 4331 course at UCF. This is a project made on the LAMP stack which allows users to add, edit, view or delete colors on a certain application. This project utilized JavaScript in order to create frontend and backend functionalities for users.

- Technologies Used:
*PHP (backend)
*MySQL (backend, database)
*HTML (frontend)
*CSS (frontend)
*JavaScript (frontend)
*LAMP Stack (Including: Linux, Apache, MySQL, PHP)

-How to Run and Access Application:
1. Please clone the repository, using this:

git clone https://github.com/Keketheedev/colors-lamp.git

2. Make a database using MySQL and utilize any SQL files (if given)

3. Change `api/config.php` with credentials of your choice

4. Start your application! 

5. Run via web browser using: 

http://localhost/colors-lamp/public/index.html

-Assumptions/Limitations/AI Usage
*There are no users implemented for your use
*Some systems may not work, please use a modern system.
*AI policy is being followed as per the COP4331 course at UCF.
